### 一、导入脚本方法
  1.将附件<user_data.conf>放到<小胖按键.exe>同级目录下，自动导入  
  2.在多按键界面新建，然后将下面脚本复制到脚本内容中保存
### 二、脚本
#### 钓鱼
```
鼠标移动|-45|10
鼠标左键|1.0|1.0
鼠标移动|45|10
等待|14
鼠标左键|1.0|1.0
等待|2
鼠标移动|80|-40
鼠标左键|0.5|0.1
等待|0.1
```
需要调整的是第四行<等待|14>, 开帮会钓鱼活动时，需要把这个时间调小一点

#### 攻防复活点挂机
```
键-按|3|1.6|1.0
鼠标移动|60|62
鼠标左键|2.0|2.0
```
不同分辨率的显示器，可能需要调整的是第二行<鼠标移动|60|62>
